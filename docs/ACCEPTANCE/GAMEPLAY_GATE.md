# Gameplay Acceptance Gate

## Non-Negotiable Gameplay Checks

Every gameplay patch must verify:

1. Sector starts.
2. Player movement works.
3. Mobile controls work.
4. A/B/Select/Start controls work.
5. Doors remain usable.
6. Automaton can be damaged and defeated.
7. Replicator Stations can be cut.
8. Gate opens only after Automaton + all Stations are down.
9. Sector-clear transition works.
10. Stats reset on next sector.

## Current Open Gameplay Regression

Burst Charge weapon behavior must be restored.

Expected:

- 8-shot magazine.
- 3-second reload only when empty.
