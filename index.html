FPS-002

DIGITAL REALITY RUN

You are moving through a digital reconstruction of reality — real rooms rebuilt from system memory and held together by a hostile command signal. Cut the signal: collapse the Automaton's command network and its Replicator Stations until the Signal Gate validates, then advance to the next sector.

PHONE  Left D-pad: ▲ forward / ▼ back / ◀▶ turn · Right A/B/X/Y · SEL cycle · START pause/resume

KEYBOARD
W/↑ forward · S/↓ back · A/← turn left · D/→ turn right
U strafe left · I strafe right · J/Space = A (fire · open door · enter Signal Gate)
K = B (Signal Baton — cut signal · pick up) · Tab = Select (cycle weapon)
Enter/P = Start (pause/resume) · R = restart sector

Automatons only. No L/R, no triggers, no mouse-look.

PAUSED
